The Challenge

Why CNN-Based Weed Detection Falls Short

Supervised deep learning approaches require thousands of labeled images, fail to generalize across regions and growth stages, and collapse entirely when confronted with weed species they have never seen.

📦

Massive Training Data Requirements

CNNs demand thousands of labeled images per crop type, per growth stage, per geographic region — a prohibitive cost for most farmers.

🌍

Geography-Locked Models

The same crop species looks visually different under different lighting, soil color, and climate conditions. A model trained in one region fails in another.

🌱

Growth Stage Drift

A corn seedling at week 2 looks nothing like the same plant at week 8. Stage-specific models multiply the data collection burden exponentially.

❓

Unknown Weed Species

New or invasive weed species that were never in the training set simply cannot be detected — a critical failure mode in real fields.

The Approach

Shape-Based Semi-Supervised Classification

WeedOut bypasses all of these limitations by classifying crops based on their structural shape, not their visual appearance. Shape is invariant to lighting, color, geography, and growth stage. Unknown weeds are handled naturally — anything that doesn't match the primary crop cluster is a weed, regardless of species.

1

Background Removal

Convert to HSV color space and apply green-range thresholding to isolate all vegetation from soil. Resize to 250×250px for efficiency.

HSV Thresholding
2

Crop Identification

Two-Pass Connected Component Labeling assigns each distinct plant a unique label. Two passes resolve equivalences so touching blobs share one ID.

Connected Component Labeling
3

Profile Plot Generation

For each plant, compute a 1D signal of pixel-width vs. height — its structural silhouette. Interpolate all plots to 250 points for uniform comparison.

Profile Plots
4

DTW-Based Clustering

Dynamic Time Warping computes pairwise shape distances between all profile plots. Low-distance pairs are merged into clusters iteratively until convergence.

Dynamic Time Warping
5

Farmer Labels Primary Crop

The farmer sees one sample image from each cluster plus its percentage of total vegetation. One selection — "this is my crop" — is all that's needed.

Semi-Supervised
6

Autonomous Spraying

Weeds are colored red, crops green. The output image — scaled to real farm dimensions — drives the ground rover, which sprays herbicide at every red pixel.

A-CPS Actuation

Key Innovations

What Makes WeedOut Different

🏋️

Zero Training Data

No labeled images, no model training, no dataset collection. WeedOut works out of the box on any farm, any crop, any geography.

🌿

Universal Generalization

Shape is invariant to lighting, soil color, growth stage, and geographic location. Works on crops and weeds never seen before.

👨‍🌾

Minimal Farmer Effort

Only one decision required: which cluster image looks like your crop. Everything else — classification, weed pressure, sprayer control — is automatic.

📱

Edge-Deployable

Only two image passes required. Runs efficiently on a smartphone or tablet in the field without cloud infrastructure.

🔴

Unknown Weed Detection

Any plant not matching the primary crop cluster is labeled a weed — automatically handling invasive or novel species the system has never encountered.

📊

Weed Pressure Reporting

Outputs weed types present, their individual percentage of total vegetation, and an overall weed pressure score — actionable intelligence beyond just classification.

Quick Start

Run WeedOut

Clone the repo, install dependencies, and run the Jupyter notebook on your field images.

# Clone the repository
git clone https://github.com/kirankkethineni/WeedOut.git
cd WeedOut

# Install dependencies
pip install opencv-python numpy matplotlib scipy pandas scikit-learn

# Launch the notebook
jupyter notebook WeedOut.ipynb

# Place your field image as finalc.jpg in the project root,
# then run all cells. At the clustering step, identify which
# cluster is your primary crop — that's the only input needed.

Dataset used in paper: Kaggle Crop and Weed Detection Dataset ↗
